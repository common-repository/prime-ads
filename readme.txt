== Description ==

Wordpress side of advertising monetization service Prime Ads (http://primeads.ru). It allows you to display a different set of ad units on a site without interfering with the code. All management takes place on the service side.

== Installation ==

1. Copy the folder `prime-ads` into the directory `wp-content/plugins/` and activate the plugin.
2. Go to http://primeads.ru and register.
3. All other information you receive after registration in your account.
